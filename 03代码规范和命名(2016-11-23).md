### **2.以小写字母开头***
- 这样可以避免与**内建类型**产生冲突
- 有**三个特殊情况**：
```javascript
//普通
var age = 35;

//DOM节点
var $body = $(document.body);

//构造函数
function MyType(){};
var obj = new MyType();

//构造函数中私有属性和方法
function MyType(){
    var _name;
    var _lock = function(){};
}
```
